Bienvenue dans le manuel d'utilisation de notre application et de notre jeu de l'oie revisité. Vous trouverez ci-dessous comment obtenir l'application, comment fonctionne l'application et pour finir comment jouer et le fonctionnement de notre jeu.
## Comment obtenir notre application ?
Avant toute chose assurez-vous d'avoir démarré votre ordinateur sous Linux (si vous utilisez les ordinateurs de l'IUT). Si ce n'est pas le cas redémarrer le dessus. Ensuite pour commencer vous devez vous rendre sur le site web disponible à l'adresse suivante (https://adminsae.alwaysdata.net/signup). Une fois que vous arrivez sur le site vous êtes sur notre page d'inscription, inscrivez-vous en utilisant de préférence avec votre adresse amu.

## Comment lancer le jeu ainsi que son fonctionnement.

Pour lancer le jeu cliqué sur le bouton Start Game.

<p align="center">
  <img width="700" height="350" src="https://user-images.githubusercontent.com/92721333/229608914-f0d8b7c5-d6c3-413c-b54f-8105926dbf4b.png">
</p>

Vous arrivez ensuite sur la page ci-dessous. 

<p align="center">
  <img width="700" height="350" src="https://user-images.githubusercontent.com/92721333/229609278-e64bf4b2-a60d-417e-a964-ef6c68879dec.png">
</p>

Le jeu se joue entre 2 à 4 joueurs.
Pour jouer il faut que vous vous connectiez avec les mêmes identifiants que sur le site web.

<p align="center">
  <img width="800" height="450" src="https://user-images.githubusercontent.com/92721333/229609646-c284eedc-f5d3-4a87-bf88-37bbda5b0485.png">
</p>
Les joueurs connectés s'afficheront sur le tableau présent sur la droite de la page.
Une fois que vous avez minimum deux joueurs de connecté vous pouvez lancer le jeu en cliquant sur le bouton "start the game".

<p align="center">
  <img width="800" height="450" src="https://user-images.githubusercontent.com/92721333/229610388-7ee19c8b-93d0-480f-9109-6ee22a9e5110.png">
</p>

<p align="center">
  <img width="800" height="450" src="https://user-images.githubusercontent.com/92721333/229611449-51a81ba0-2b4b-4e6e-8c82-f88c9b9859f7.png">
</p>

Je jeu est un tour par tour, il y a trois types de difficulté pour les questions reliées aux cases (vert = facile, jaune = moyen et rouge = difficile).
Une fois le dé lancer vous avez 20 secondes pour répondre. Votre pion avance seulement si votre réponse est juste.

Consultez la notice au besoin. Bon jeu à vous.
