# Goose Game
Bienvenue dans le manuel d'utilisation de notre application et de notre jeu de l'oie revisité. Vous trouverez ci-dessous comment obtenir l'application, comment fonctionne l'application et pour finir comment jouer et le fonctionnement de notre jeu.
## Comment obtenir notre application ?
Avant toute chose assurez-vous d'avoir démarré votre ordinateur sous Linux (si vous utilisez les ordinateurs de l'IUT). Si ce n'est pas le cas redémarrer le dessus.
Ensuite pour commencer vous devez vous rendre sur le site web disponible à l'adresse suivante (https://adminsae.alwaysdata.net/signup)
Une fois que vous arrivez sur le site vous êtes sur notre page d'inscription, inscrivez-vous en utilisant de préférence avec votre adresse amu.

</p>

## Comment lancer notre application ?
Lancer Intellij Idea. (Si vous utilisez les ordinateurs de l'IUT il faut démarrer sous Linux).

Une fois sur Intellij Idea ouvrez le dossier qui contient notre application. Pour ce faire cliquez sur File puis open et séléctionnez le dossier que vous venez d'extraire.
<p align="center">
  <img width="200" height="200" src="https://user-images.githubusercontent.com/92721333/212495706-eee9b360-d358-4e61-b6ae-9ec5c002b066.JPG">
</p>

Une fois le projet ouvert, vous pouvez retrouver sur la gauche différente classe, double-cliquez sur la classe "AccueilMain".
Sur votre écran sera affiché le code de la classe "AccueilMain", appuyer sur le triangle vert présent sur l'image ci-dessous pour lancer l'application.

<p align="center">
  <img width="400" height="200" src="https://user-images.githubusercontent.com/92721333/212558517-cc6029a2-b7e4-4df3-8ca0-e27d1bdbfe5b.JPG">
</p>

Une fois que vous avez lancé la classe AccueilMain, le terminal s'ouvre et vous obtenez ceci :

<p align="center">
  <img width="450" height="250" src="https://user-images.githubusercontent.com/92721333/212558787-e8fbe708-4c4a-4b99-9440-294a95f23dc4.JPG">
</p>

Ici il faut que vous cliquiez à côté de la flèche puis entrez le nom du joueur1 et valider puis pareil pour le joueur 2.

<p align="center">
  <img width="450" height="250" src="https://user-images.githubusercontent.com/92721333/212558864-bf0333b1-3d42-44d2-9e57-515ed15763ef.JPG">
</p>

Ensuite deux choix s'offrent à vous :
- si vous saisissez 1 l'application se lancera pour 2 joueurs.
<p align="center">
  <img width="450" height="250" src="https://user-images.githubusercontent.com/92721333/212559077-ece4fecf-4b52-48d2-a83a-a4d903e1df64.JPG">
</p>
- Si vous saisissez 2 vous pourrez rentrer le nom d'un 3éme joueurs. 
Et ce allant jusqu'à 4joueurs maxmimum.
<p align="center">
  <img width="450" height="250" src="https://user-images.githubusercontent.com/92721333/212559031-b818b68a-b842-49ce-b9ff-9d7e52b7ff8a.JPG">

Une fois sur l'application nous vous recommandons de prendre connaissance de la notice qui vous décrit les règles du jeu. Pour ce faire il vous suffit de cliquer sur le bouton Notice.
